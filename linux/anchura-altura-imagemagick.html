Saber la anchura y altura de una imagen con ImageMagick

Para saber información de una imagen con ImageMagick, podemos usar el comando identify:

identify imagen.jpg

Este comando ofrece información detallada sobre la imagen, incluyendo su tamaño, formato, profundidad de bits, y más.

Para saber el tamaño de una imagen con ImageMagick, podemos usar el comando identify con el modificador -format %w para obtener la anchura:

identify -format %w imagen.jpg

O con el modificador -format %w para obtener la altura:

identify -format %h imagen.jpg

También podemos usar estos comandos en un script de Bash para trabajar con los valores de anchura y altura de una imagen:

#!/bin/bash
imagen="imagen.jpg"
anchura=$(identify -format %w "$imagen")
altura=$(identify -format %h "$imagen")
echo "La imagen $imagen mide $anchura píxeles de anchura y $altura píxeles de altura."
