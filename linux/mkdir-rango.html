Crear directorios con un rango de nombres (de la a a la z)

Para crear directorios con un rango de nombres, por ejemplo, de la a a la z, podemos usar un bucle for en Bash para crear directorios en ese rango.

for letter in {a..z}; do
  mkdir ./$letter
done

Mover los ficheros que empiezan por una letra a su respectivo directorio

Para mover los ficheros que empiezan por cada letra del alfabeto a su respectivo directorio, podemos usar un bucle for junto con el comando mv. Si queremos que sólo mueva los ficheros y no los directorios, podemos comprobarlo con if [ -f ].

for letter in {a..z}; do
  mkdir -p ./$letter
  for file in ./$letter* ./${letter^^}*; do
    if [ -f "$file" ]; then
      mv "$file" ./$letter/
    fi
  done
done

Este script hace lo siguiente:

Crea el directorio correspondiente si no existe (mkdir -p).

Mueve los ficheros que empiezan por la letra en minúscula (mv ./$letter* ./$letter/).

Mueve los ficheros que empiezan por la letra en mayúscula (mv ./${letter^^}* ./$letter/).

Verifica si el elemento es un fichero (-f "$file").

Redirige cualquier error a /dev/null para evitar mensajes de error si no hay ficheros que mover (2>/dev/null).

for letter in {a..z}; do
  mkdir -p ./$letter
  mkdir -p ./"$letter"/borrar
  mkdir -p ./"$letter"/generarpose
  mkdir -p ./"$letter"/modificar
  for file in ./$letter* ./${letter^^}*; do
    if [ -f "$file" ]; then
      mv "$file" ./$letter/
    fi
  done
  mkdir -p ./misc
  mkdir -p ./misc/borrar
  mkdir -p ./misc/generarpose
  mkdir -p ./misc/modificardone
  for file in ./*; do
    if [ -f "$file" ]; then
      mv "$file" ./misc/
    fi
  done
  rmdir *
