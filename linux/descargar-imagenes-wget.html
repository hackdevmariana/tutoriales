Cómo descargar únicamente las imágenes de un sitio web con el comando wget

Para descargar únicamente las imágenes de un sitio web, podemos usar wget con ciertos modificadores:

• -r: Activa la descarga recursiva.

• -l1: Limita la profundidad de la descarga a 1 nivel (podemos ajustar este valor según las necesidades que tengamos).

• -A jpg,jpeg,png,gif: Especifica las extensiones de ficheros que deseamos descargar (en este caso, imágenes).

• -nd: Evita la creación de directorios, guardando todos los archivos en el mismo directorio.

• -P /ruta/a/tu/directorio: Especifica el directorio donde se guardarán las imágenes.

• -e robots=off: Ignora el archivo robots.txt del sitio web, permitiendo la descarga de todos los archivos especificados.

Por lo tanto, el comando quedaría así:

wget -r -l1 -A jpg,jpeg,png,gif -nd -P /ruta/a/tu/directorio -e robots=off http://sitioweb.com

Este comando descargará todas las imágenes con las extensiones especificadas desde el sitio web indicado y las guardará en el directorio que hayamos especificado.
