Crear múltiples directorios con una línea en Bash

Para crear múltiples directorios con un único comando en Linux, podemos usar un bucle for en combinación con el comando mkdir.

Por ejemplo, para crear directorios de la a a la z, podemos hacerlo con:

for dir in {a..z}; do mkdir "$dir"; done

Este comando utiliza un bucle for para iterar a través de las letras a a z, creando un directorio para cada letra.

Si lo que queremos es crear directorios con un prefijo y un sufijo de la a a la z:

for dir in {a..z}; do mkdir "dir_${dir}"; done

Este comando es similar al anterior, pero añade un prefijo dir_ a cada directorio.
