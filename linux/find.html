Comando find

El comando find en Linux permite buscar ficheros y directorios en el sistema de ficheros.

Permite realizar búsquedas basadas en varios criterios como el nombre, el tamaño, la fecha de modificación, los permisos..., entre otros.

Buscar ficheros por nombre

find /ruta/del/directorio -name "nombre_del_archivo"

Este comando buscará ficheros con el nombre especificado en el directorio y sus subdirectorios.

Buscar ficheros por tipo

find /ruta/del/directorio -type f

Esto buscará solo ficheros (no directorios) en el directorio especificado.

Buscar ficheros por tamaño

find /ruta/del/directorio -size +100M

Este comando buscará ficheros mayores a 100 MB.

Buscar ficheros modificados en los últimos 7 días

find /ruta/del/directorio -mtime -7

Esto buscará ficheros que han sido modificados en los últimos 7 días.

Ejecutar un comando en los ficheros encontrados

find /ruta/del/directorio -name "*.log" -exec rm {} \;

Este comando buscará ficheros con extensión .log y los eliminará.

Buscar por nombre y mover ficheros

Para buscar todos los ficheros cuyo nombre contenga la subcadena "intro" a partir del directorio actual y sus subdirectorios, y moverlos a un directorio llamado ./intro/, puedes usar el siguiente comando:

find . -type f -name "*intro*" -exec mv {} ./intro/ \;

Buscar ficheros por su contenido

Para buscar ficheros por su contenido desde la consola, podemos combinar el comando find con el comando grep:

find /ruta/del/directorio -type f -exec grep -l "texto_a_buscar" {} \;

Este comando hará lo siguiente:

  1. find /ruta/del/directorio -type f: Busca todos los ficheros (-type f) en el directorio especificado y sus subdirectorios.

  2. -exec grep -l "texto_a_buscar" {}: Por cada archivo encontrado, ejecuta el comando grep para buscar el texto especificado. La opción -l hace que grep solo muestre los nombres de los ficheros que contienen el texto.

  3. \;: Indica el final del comando -exec.

Por ejemplo, si queremos buscar ficheros que contengan la palabra "error" en el directorio /var/log, usarías:

find /var/log -type f -exec grep -l "error" {} \;

Uso de operador OR

Para usar el operador lógico OR en el comando find, puedes utilizar la opción -o. Esto te permite combinar múltiples condiciones y encontrar ficheros que cumplan al menos una de ellas. Aquí tienes un ejemplo:

find /ruta/del/directorio \( -name "*.txt" -o -name "*.log" \)

Este comando buscará ficheros que tengan la extensión .txt o .log en el directorio especificado y sus subdirectorios.

Si queremos combinar más condiciones, simplemente añade más -o entre ellas. Por ejemplo, para buscar ficheros con extensiones .txt, .log o .conf:

find /ruta/del/directorio \( -name "*.txt" -o -name "*.log" -o -name "*.conf" \)

Uso de operador AND

El operador lógico AND en el comando find se utiliza implícitamente, ya que find evalúa todas las condiciones especificadas y solo devuelve los ficheros que cumplen con todas ellas. No necesitas un operador especial para AND, simplemente puedes encadenar las condiciones.

Aquí tienes un ejemplo:

find /ruta/del/directorio -type f -name "*.txt" -size +1M
