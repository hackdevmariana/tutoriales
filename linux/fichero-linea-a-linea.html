Cómo recorrer línea a línea un fichero en Bash

Para recorrer línea a línea un fichero en Bash, puedes usar un bucle while junto con el comando read. Veamos las partes clave del script:

• IFS=: Establece el separador de campo interno a una cadena vacía para preservar los espacios en blanco al principio y al final de cada línea.

• read -r linea: Lee cada línea del fichero y la almacena en la variable linea.

• done < "$fichero": Redirige la entrada del bucle while al fichero especificado.

El script quedaría así:

#!/bin/bash

fichero="ruta/al/fichero.txt"

while IFS= read -r linea
do
  echo "$linea"
done < "$fichero"

Este script imprimirá cada línea del fichero en la salida estándar. Podemos reemplazar echo "$linea" con cualquier otro comando que queramos ejecutar para cada línea del fichero.
