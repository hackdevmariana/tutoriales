Borrar un fichero de imagen si su anchura o altura es menor a un determinado número de píxeles

Para borrar los ficheros de imagen de un directorio cuyo tamaño (altura o anchura) sea menor a uno especificado, podemos comprobar el tamaño de la imagen usando identify de ImageMagick, y borrar los ficheros con rm.

#!/bin/bash

directorio="/ruta/al/directorio/de/imagenes"

anchura_minima=100
altura_minima=100

for archivo in "$directorio"/*; do
  if [[ $archivo == *.jpg || $archivo == *.jpeg || $archivo == *.png || $archivo == *.gif ]]; then
    anchura=$(identify -format %w "$archivo")
    altura=$(identify -format %h "$archivo")

    if (( anchura < anchura_minima || altura < altura_minima )); then
    echo "Borrando $archivo de tamaño ${anchura}x${altura} píxeles"
    rm "$archivo"
    fi
  fi
done
