Ejecutar múltiples veces un comando

Para ejecutar múltiples veces un comando, podemos usar un bucle for en la línea de comandos.

Por ejemplo, para ejecutar un comando cien veces, podemos usar:

for i in {1..100}; do comando; done

Si queremos usar parámetros, también podemos hacerlo:

for i in {1..100}; do applyeffect.py prueba image.jpg; done
