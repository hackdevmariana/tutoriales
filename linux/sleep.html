Cómo hacer una pausa con sleep en Bash

Para hacer una pausa en un script de Bash, podemos usar el comando sleep seguido del tiempo que queramos que el script espere. Podemos especificar el tiempo en segundos (s), minutos (m), horas (h) o días (d).

Por defecto, sleep mide el tiempo en segundos. Así, para esperar 5 segundos:

sleep 5

Para esperar 2 minutos:

sleep 2m

Para esperar 1 hora:

sleep 1h

Para esperar 1 día:

sleep 1d
