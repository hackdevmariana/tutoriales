Web scraping en R

Uno de los paquetes más comunes y poderosos para el web scraping es rvest.

Instalar y cargar el paquete rvest:

install.packages("rvest")
library(rvest)

Especificar la URL de la página web que deseamos scrapear

url <- "https://www.ejemplo.com"

Leer el HTML de la página web:

pagina <- read_html(url)

Extraer el texto de un elemento específico (por ejemplo, el título):

titulo <- pagina %>% html_nodes("h1") %>% html_text()

Extraer los enlaces de la página:

enlaces <- pagina %>% html_nodes("a") %>% html_attr("href")

Extraer tablas de la página:

tablas <- pagina %>% html_nodes("table") %>% html_table()

Extraer información de elementos específicos utilizando selectores CSS. Por ejemplo, si tenemos un elemento con la clase "descripcion" en un div con la clase "producto":

descripcion <- pagina %>% html_nodes("div.producto .descripcion") %>% html_text()

Una vez que hayamos extraído los datos de la página web, podemos manipularlos y analizarlos según las necesidades del proyecto utilizando las funciones y herramientas de análisis de datos en R.
