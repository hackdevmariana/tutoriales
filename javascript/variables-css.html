Trabajar con variables CSS desde JavaScript

Cambiar el valor de una variable CSS

Para cambiar el valor de una variable CSS con JavaScript, tenemos la propiedad style.setProperty().

Veamos un ejemplo paso a paso:

Primero, definamos una variable en un archivo CSS:

:root {
  --main-color: #3498db;
}

Ahora ya podemos acceder y modificar esta variable con JavaScript.

Primero, seleccionamos el elemento. En este ejemplo, el elemento raíz (root).

var root = document.documentElement;

Una vez seleccionado, ya podemos cambiar el valor de la variable:

root.style.setProperty('--main-color', '#e74c3c');

En este ejemplo, la variable --main-color se cambia de #3498db a #e74c3c.

Acceder al valor de una variable

Para obtener el valor actual de una variable CSS con JavaScript, podemos usar el método getComputedStyle() junto con getPropertyValue().

Siguiendo con el ejemplo anterior, si queremos saber el valor actual de la variable --main-color y trabajar con él, primero debemos seleccionar el elemento con el que operar, en este ejemplo, con la raíz:

var root = document.documentElement;

Y, ahora, ya podemos obtener el valor de la variable:

var mainColor = getComputedStyle(root).getPropertyValue('--main-color');

console.log(mainColor); // Salida: #3498db

En este ejemplo, guardaremos en la variable JS mainColor el valor de la variable CSS --main-color y la mostraremos con console.log().
