Crear y recorrer una lista en PHP

Podemos crear una lista en PHP utilizando array:

$colors = array('rojo', 'verde', 'azul');

Y recorrer los elementos de la lista usando foreach:

foreach ($colors as $color) {
  echo $color . "<br>";
}
