Crear y recorrer una lista multidimensional

Una lista multidimensional en PHP es, básicamente, un array que contiene otros arrays.

Por lo tanto, para crear una lista multidimensional, usaremos array dentro de otro array:

<?php
$productos = array(
  array("nombre" => "Bolígrafo", "precio" => 0.75, "cantidad" => 100),
  array("nombre" => "Cuaderno", "precio" => 2.50, "cantidad" => 50),
  array("nombre" => "Borrador", "precio" => 0.50, "cantidad" => 200)
);
?>

En este ejemplo, hemos creado un array llamado $productos que contiene tres arrays, cada uno representando un producto con sus respectivos detalles: nombre, precio y cantidad.

Para acceder a los elementos del array multidimensional, podemos acceder por su índice. Si lo que queremos es recorrer el array, usaremos foreach:

<?php

foreach ($productos as $producto) {
  echo "Nombre: " . $producto["nombre"] . "<br>";
  echo "Precio: " . $producto["precio"] . "<br>";
  echo "Cantidad: " . $producto["cantidad"] . "<br><br>";
}
?>

Si necesitamos trabajar con arrays de más dimensiones, simplemente anidaremos más arrays dentro de los arrays existentes.

Veamos un ejemplo de un array tridimensional:

<?php
$almacen = array(
  "Sección 1" => array(
    "Estante 1" => array("Producto A", "Producto B"),
    "Estante 2" => array("Producto C", "Producto D")
  ),
  "Sección 2" => array(
    "Estante 1" => array("Producto E", "Producto F"),
    "Estante 2" => array("Producto G", "Producto H")
  )
);
?>

Y cómo acceder a los elementos del array tridimensional:

<?php
foreach ($almacen as $seccion => $estantes) {
  echo "En " . $seccion . ":<br>";
  foreach ($estantes as $estante => $productos) {
    echo "  En " . $estante . ":<br>";
    foreach ($productos as $producto) {
      echo "    " . $producto . "<br>";
    }
  }
}
?>
