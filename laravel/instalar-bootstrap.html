Instalar Bootstrap en un proyecto Laravel

Veamos los pasos a seguir para instalar y poder utiliar Bootstrap en un proyecto Laravel:

Instalar Laravel UI

Laravel UI es un paquete que proporciona plantillas de autenticación y soporte para Bootstrap.

Primero, debemos estar en el directorio del proyecto Laravel (importante) y luego ejecutar el siguiente comando:

composer require laravel/ui

Generar plantillas de Bootstrap

Una vez instalado Laravel UI, podemos generar las plantillas de Bootstrap con el siguiente comando:

php artisan ui bootstrap

Si también necesitamos autenticación, podemos añadir --auth al comando:

php artisan ui bootstrap --auth

Instalar dependencias de npm

A continuación, instalaremos las dependencias de NPM, incluyendo Bootstrap:

npm install

Compilar los archivos CSS y JS

Compilamos los archivos CSS y JS usando Laravel Mix:

npm run dev

Uso de Bootstrap en Laravel

Incluir Bootstrap en las vistas

Para asegurarnos de que los archivos compilados se incluyan en las vistas, normalmente, se hace en el archivo resources/views/layouts/app.blade.php:

<!DOCTYPE html>
  <html lang="{{ str_replace('_', '-', app()->getLocale()) }}">
    <head>
      <!-- Otros metadatos -->
      <link href="{{ mix('css/app.css') }}" rel="stylesheet">
    </head>
    <body>
      <div id="app">
      <!-- Contenido de la aplicación -->
      </div>
      <script src="{{ mix('js/app.js') }}"></script>
    </body>
  </html>

Personalizar Bootstrap (Opcional)

Si deseamos personalizar Bootstrap, podemos editar el fichero resources/sass/app.scss para importar sólo los componentes que necesitemos o para añadir variables y estilos personalizados.

@import '~bootstrap/scss/bootstrap';
