Escribir en una imagen con PILLOW

Para escribir en una imagen con Pillow, necesitaremos las clases Image, ImageDraw e ImageFont:

from PIL import Image, ImageDraw, ImageFont

Cargaremos la imagen:

img = Image.open("path/to/image.jpg")

Crear con esa imagen un objeto ImageDraw:

draw = ImageDraw.Draw(img)

Debemos definir la tipografía y el tamaño del texto:

font = ImageFont.truetype("arial.ttf", 40)

Escribimos el texto en la imagen con el método text() definiendo:

  • Posición de las letras

  • Texto a escribir

  • Tipografía

  • Color del texto

draw.text((10, 10), "Hola, mundo!", font=font, fill=(255, 255, 255))

Una vez procesada la imagen, la podemos guardar con el método save() de la clase Image:

img.save("path/to/image_with_text.jpg")

Script completo:

from PIL import Image, ImageDraw, ImageFont
img = Image.open("path/to/image.jpg")
draw = ImageDraw.Draw(img)
font = ImageFont.truetype("arial.ttf", 40)
draw.text((10, 10), "Hola, mundo!", font=font, fill=(255, 255, 255))
img.save("path/to/image_with_text.jpg")
