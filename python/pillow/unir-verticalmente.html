Unir verticalmente dos imágenes

Importamos PILLOW:

from PIL import Image

Abrimos las dos imágenes:

imagen1 = Image.open('path/to/img_1.jpg')
imagen2 = Image.open('path/to/img_2.jpg')

Calculamos el ancho y la altura total:

ancho_total = max(imagen1.width, imagen2.width)
altura_total = imagen1.height + imagen2.height

Creamos una nueva imagen en blanco con el tamaño adecuado:

imagen_unida = Image.new('RGB', (ancho_total, altura_total))

Pegamos las dos imágenes una debajo de la otra:

imagen_unida.paste(imagen1, (0, 0))
imagen_unida.paste(imagen2, (0, imagen1.height))

Guardamos la imagen resultante:

imagen_unida.save('path/to/joined_images.jpg')
