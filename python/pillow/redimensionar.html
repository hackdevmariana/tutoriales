Redimensionar imágenes con Python y PILLOW

Importar PILLOW:

from PIL import Image

Abrir una imagen existente:

imagen = Image.open("ruta/a/tu/imagen.jpg")

Definir el nuevo tamaño (con una tupla):

nuevo_tamano = (800, 600)

Cambiar el tamaño de la imagen:

imagen_redimensionada = imagen.resize(nuevo_tamano, Image.ANTIALIAS)

Guardar la imagen redimensionada:

imagen_redimensionada.save("ruta/a/tu/imagen_redimensionada.jpg")

Script completo:

from PIL import Image
imagen = Image.open("ruta/a/tu/imagen.jpg")
nuevo_tamano = (800, 600)
imagen_redimensionada = imagen.resize(nuevo_tamano, Image.ANTIALIAS)
imagen_redimensionada.save("ruta/a/tu/imagen_redimensionada.jpg")
