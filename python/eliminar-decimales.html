Cómo eliminar decimales en Python

Para eliminar los decimales de un número en Python, podemos convertir el número a un entero usando la función int().

numero_con_decimales = 123.456
numero_sin_decimales = int(numero_con_decimales)

print(numero_sin_decimales)

Salida: 123

Otra opción es usar la función math.floor() o math.ceil() si queremos redondear hacia abajo o hacia arriba, respectivamente

import math

numero_con_decimales = 123.456

Redondear hacia abajo:

numero_redondeado_abajo = math.floor(numero_con_decimales)
print(numero_redondeado_abajo)

Salida: 123

Redondear hacia arriba:

numero_redondeado_arriba = math.ceil(numero_con_decimales)
print(numero_redondeado_arriba)

Salida: 124
