Módulo imageio

Imageio es una biblioteca de Python que proporciona una interfaz sencilla para leer y escribir una amplia gama de datos de imágenes, incluyendo imágenes animadas, datos volumétricos y formatos científicos.

Soporta una variedad de formatos de archivos de imagen y video, y se integra bien con otras bibliotecas populares de procesamiento de imágenes y video, como PIL/Pillow, NumPy y OpenCV.

Cuenta con soporte multiformato, permitiendo operar con:

  • Imágenes: JPEG, PNG, BMP, GIF, TIFF, etc.

  • Videos: AVI, MP4, MKV, MOV, etc.

  • Volúmenes y secuencias de imágenes.

Es compatible con arrays de NumPy, lo que facilita la manipulación de datos de imágenes y videos y para manipulación de datos numéricos. También puede trabajar en conjunto con PIL/Pillow para operaciones de procesamiento de imágenes.

Cuenta con integración con FFmpeg para procesamiento de video avanzado.

Instalación

sh pip install imageio

Uso

Importar:

sh import imageio

Leer una imagen:

sh image = imageio.imread('path/to/image.png')

Mostrar información de la imagen:

sh print(image.shape)

Escribir una imagen (un cuadrado blanco de 100x100):

``sh import imageio import numpy as np

image = np.ones((100, 100, 3), dtype=np.uint8) * 255

imageio.imwrite('path/to/output_image.png', image) ``

Leer y recorrer vídeo fotograma a fotograma:

``sh import imageio

video = imageio.get_reader('path/to/video.mp4', 'ffmpeg')

for frame in video: print(frame.shape) ``

Escribir un vídeo con 100 frames de ejemplo:

``sh import imageio import numpy as np

video_writer = imageio.get_writer('output_video.mp4', fps=30)

for i in range(100): frame = np.random.randint(0, 255, (720, 1280, 3), dtype=np.uint8) video_writer.append_data(frame)

video_writer.close() ``
