Ordenar listas en Python

Hay varias formas de ordenar una lista en Python:

Usando el método sort()

El método sort() modifica la lista original ordenándola y no devuelve ningún valor.

mi_lista = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
mi_lista.sort()
print(mi_lista)

Salida: [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9]

Usando la función sorted()

La función sorted() devuelve una nueva lista ordenada y no modifica la lista original.

mi_lista = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
lista_ordenada = sorted(mi_lista)
print(lista_ordenada)
print(mi_lista)

Salida: [1, 1, 2, 3, 3, 4, 5, 5, 5, 6, 9] La lista original permanece sin cambios: [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]

Ordenar en orden inverso

Puedes ordenar una lista en orden inverso usando el parámetro reverse=True de sort().

mi_lista = [3, 1, 4, 1, 5, 9, 2, 6, 5, 3, 5]
mi_lista.sort(reverse=True)
print(mi_lista)

Salida: [9, 6, 5, 5, 5, 4, 3, 3, 2, 1, 1]

Ordenar una lista de objetos

Si tenemos una lista de objetos, podemos usar el parámetro key para especificar una función que extraiga un valor de comparación de cada elemento.

class Persona:
def __init__(self, nombre, edad):
self.nombre = nombre
self.edad = edad

def __repr__(self):
return f'{self.nombre} ({self.edad})'

personas = [Persona('Ana', 23), Persona('Luis', 19), Persona('Marta', 31)]
personas.sort(key=lambda persona: persona.edad)
print(personas)

Salida: [Luis (19), Ana (23), Marta (31)]
