Extraer un elemento aleatorio de una lista

Para extraer un elemento aleatorio de una lista podemos usar el método choices del módulo random:

import random
colors = ['Rojo', 'Verde', 'Azul', 'Blanco', 'Negro']

random_color = random.choices(colors)
print(f"Color aleatorio: {random_color}")

Si queremos seleccionar más de un elemento aleatorio, podemos indicar a random.choices el parámetro k con la cantidad de elementos que deseemos extraer:

import random
colors = ['Rojo', 'Verde', 'Azul', 'Blanco', 'Negro']

random_colors = random.choices(colors, k=2)
print(f"Colores aleatorios: {random_colors}")

Nos devolverá el resultado siempre en formato lista.

Para convertir una lista con un único elemento en cadena, podemos indicar el índice 0:

import random
colors = ['Rojo', 'Verde', 'Azul', 'Blanco', 'Negro']

random_color = random.choices(colors)
print(f"Color aleatorio: {random_color[0]}")

Para convertir a cadena una lista con varios elementos, podemos usar el método join:

import random
colors = ['Rojo', 'Verde', 'Azul', 'Blanco', 'Negro']

random_colors = random.choices(colors, k=2)
print(f"Colores aleatorios: {' '.join(random_colors)}")
