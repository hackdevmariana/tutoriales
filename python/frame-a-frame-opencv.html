Recorrer vídeo frame a frame con OpenCV

Para recorrer un vídeo frame a frame en Python, una de las mejores opciones es usar la biblioteca OpenCV.

import cv2

Abrir el video

video_path = 'ruta/al/video.mp4'
cap = cv2.VideoCapture(video_path)

Verificar si el video se abrió correctamente

if not cap.isOpened():
  print("Error al abrir el video")
  exit()

Recorrer el video frame a frame:

while True:
  ret, frame = cap.read()
if not ret:
  break

Mostrar el frame:

cv2.imshow('Frame', frame)

Esperar 25ms y salir del bucle si se presiona la tecla 'q':

if cv2.waitKey(25) & 0xFF == ord('q'):
  break

Liberar el objeto de captura y cerrar todas las ventanas:

cap.release()
cv2.destroyAllWindows()
