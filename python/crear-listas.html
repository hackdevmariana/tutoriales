Crear listas en Python

Una lista es uno de los distintos elementos iterables de Python. Para crear una lista en Python, debemos incluir los elementos entre corchetes:

lista = ["elemento 1", "elemento 2", "elemento 3"]

Si no indicamos elementos entre los corchetes, crearemos una lista vacía:

lista_vacía = []

Una lista puede contener cualquier tipo de variables.

Por ejemplo, números:

lista_de_numeros = [1, 2, 3, 4, 5]

O cadenas y números:

lista_mixta = [1, 2, 3, 4, 5, "uno", "dos", "tres", "cuatro", "cinco"]

Podemos convertir otro tipo de objetos en lista usando la función list():

lista_desde_tupla = list((1, 2, 3, 4, 5))

O usando list() con iterables.

Para convertir una cadena en una lista de caracteres:

listado_de_caracteres = list("hola")

También podemos crear una lista usando list() con funciones generadoras:

lista_generada = list(map(lambda x: x*2, range(5)))

Generar una lista a través de una comprensión de listas.

Para crear una lista con números del 0 al 9:

del_cero_al_nueve = [x for x in range(10)]

Podemos crear una lista con elementos iguales a través de la multiplicación de listas.

Por ejemplo, para crear una lista con cinco ceros:

cinco_ceros = [0] * 5
